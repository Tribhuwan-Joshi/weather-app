# weather-app
- This is a weather App but I give this a minecraft touch
- Most of the DOM manipulation in written in `Module Pattern` and `IIFE`

## Technologies Used
- openweather API
- Webpack 
- Tailwindcss
- Eslint
- Babel
- Html
- CSS
- JS

### Weather Forecast
- Wind Speed
- Maximum and Minimum Temperature
- Chance of Raining
